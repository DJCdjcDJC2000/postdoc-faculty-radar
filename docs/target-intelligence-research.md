# 目标导师与青年学者情报调研说明

更新日期：2026-07-09

## 调研目标

本轮不是普通职位聚合，而是为应用数学、数学优化、数值分析、科学计算、随机互补问题、变分不等式、SAA、非光滑优化等方向建立长期职业情报库。

情报库分两类：

- 目标导师/课题组：优先 QS 2027 Top 50 高校，尤其欧洲、香港、新加坡；记录导师主页、课题组主页、openings 或 fellowship host 信号、研究匹配理由和代表作。
- 青年学者路径样本：记录已经在做 postdoc、进入高校任职或拿到重要基金/奖项的相关青年学者；记录背景、路径、代表作、主页和可学习点。

## 首批收录规则

- 学校范围：优先 QS 2027 Top 50。
- 地区优先级：香港、新加坡、欧洲优先；北美/全球顶校后续作为 watch 补充。
- 方向门槛：必须命中具体数学/优化/数值/科学计算信号，不能仅凭 AI、machine learning、data science 入库。
- 证据要求：每条必须至少有一个官方主页、个人主页、院系主页、openings 页面、QS 页面、Google Scholar 或公开论文链接。
- 招聘状态：区分 `active_openings`、`active_department_postdoc`、`watch_profile`、`watch_young_group`、`watch_historical_postdoc`、`fellowship_host_watch`。
